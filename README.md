# SaaSpasse

La bande dessinée interactive du SaaSpaladin.

## Technologies

- HTML5
- CSS3 (vanille)
- JavaScript (vanille)

## Fonctionnalités

- Navigation par clic sur desktop
- Navigation par swipe sur mobile
- Navigation au clavier (flèches)
- Design responsive
- Léger et rapide

## Développement

Ouvrir `index.html` dans un navigateur ou utiliser un serveur local.

## Déploiement

Hébergé sur Netlify.
